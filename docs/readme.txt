使用说明

1、员工工资表、经纪人工作表直接取自HR系统导出，导出来的员工工资表需填列个人社保分险种数据；社保公积金表使用统一设置的模板；
2、工资社保表excel文件，存放“员工工资表”的工作表Sheet,取名为“员工”；存放“经纪人工资表”的工作表Sheet,取名为“经纪人”；存放“社保公积金表”的工作表Sheet,取名为“社保”。
3、员工工资表的科目代码标题行必须位于第4行，“人员编码:人员类别”列必须位于第3列、即原来存放营业部名称的“所属营业部”所在列；
   经纪人工资表的科目代码标题行也必须位于第4行，“人员编码:人员类别”列必须位于第2列、即原来存放营业部名称的“部门”所在列；
   “人员编码:人员类别”其间的冒号:是英文状态下的冒号:
4、科目代码标题行(可用于工资表中插入)、社保公积金表统一模板见工资表科目代码表头.xlsx文件。
